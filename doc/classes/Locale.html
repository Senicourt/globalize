This is the default Translator. It simply looks for all .rb files in RAILS_ROOT/config/locale/ and eval() them. It also has support for productized apps and if SITE_ROOT is defined it will overwrite default values from RAILS_ROOT with those from SITE_ROOT.

If you want to write your own translator, for example fetching data from a database or reading gettext files instead, you only need to implement the method load_locale_data(locale) which fills the @locales hash with data.
